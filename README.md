# African Startups WebScraper

## Description
This repo is a webapp that lets users collect data related to startups from more or less any website who lists startups information.
The app uses Puppeteer as a headless browser, Express for the backend API as well as html, Bootstrap and JQuery for the UI.

## Run Locally

To run locally:

1. Clone repo: https://github.com/YounesAEO/webscraper.git
2. Install backend dependencies: `npm install`
3. Add backend .env files with the correspondant environment variables, see .env.example files.
4. `npm run dev` to run the backend
